# TuneConverter

数字谱转调工具
- 支持升降八度、半音
- 支持升降号在左右侧的格式
- 支持括号合并
- 支持“//”和“/* */”注释文本

Xamarin版更换谱子之后请点击【转换】按钮，再使用其它功能，不然转换的还是之前的谱子
